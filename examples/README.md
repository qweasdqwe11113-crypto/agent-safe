# Examples

这个目录用于存放项目演示时可以直接展示的输入输出样例。

当前包含两组主要样例：

- `plain-input.txt`
  普通文本输入样例
- `plain-redacted.txt`
  对应的普通文本脱敏输出
- `plain-token-map.json`
  普通文本脱敏后生成的 token 映射
- `json-input.json`
  JSON / 嵌套对象输入样例
- `json-redacted.json`
  对应的 JSON 递归脱敏输出
- `json-token-map.json`
  JSON 样例脱敏后生成的 token 映射

这几组文件可以直接用于 README 演示、课程汇报截图或后续手动验证。
